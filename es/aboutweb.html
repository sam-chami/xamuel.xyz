¿Cómo se creó este sitio web?

Este sitio web utiliza principalmente HTML5 y CSS. Solo en casos específicos, como en The Cave, se utiliza Javascript. Puedes obtener más información sobre cómo se hizo The Cave aquí.

¿Dónde se aloja este sitio web?

xamuel.xyz está alojado de forma gratuita en las páginas de GitHub, esto es lo que dicen sobre la recopilación de datos.

When a GitHub Pages site is visited, the visitor's IP address is logged and stored for security purposes, regardless of whether the visitor has signed into GitHub or not. For more information about GitHub's security practices, see GitHub Privacy Statement.

Para DNS uso Cloudflare. Esto es lo que tienen que decir sobre la recopilación de datos:

For the purposes of mitigating DDoS attacks and providing traffic visibility through the Network Analytics dashboard, Cloudflare collects data from different protocols such as IP, IPv6, TCP, UDP, ICMP, GRE, and DNS.

¿Usas Google Analytics?

No, utilizo una alternativa a Google Analytics que respeta la privacidad y está alojada de forma independiente, llamada Plausible Analytics. Plausible no utiliza cookies ni otros tipos de identificadores personales. Solo realiza un seguimiento del uso general del sitio, no de individuos. Mi instancia de Plausible Analytics está alojada por mí mismo en España.

Puedes consultar las estadísticas y métricas de este sitio web aquí: insights.xamuel.xyz/xamuel.xyz
