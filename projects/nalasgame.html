Nala's Game

Nala's Game is a simple retro game for the Game Boy.

In the game, you have to catch different humans. The humans will try to run away from you, each at a different speed. If any human succeeds in getting to their home, you'll lose.

I developed this game in C with the gbdk-2020 sdk after seeing some of the tutorials by Gaming Monsters. (Which are great honestly).
